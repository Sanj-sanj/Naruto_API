How to use:

To get a all quotes off one big JSON, you can either type "/all" at the end of this page's URL, or click: here.

To get a random quote you can either type "/random" at the end of this page's URL, or click: here.

To search a specific quote; type after the URL "/search/{_id}" and specifiy the id parameter. Ex: /search/30FfXJRRjO
