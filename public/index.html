Welcome to my Naruto quotes API!

How to use:

To get all quotes off one big JSON, you can either type "/api/all" at the end of this page's URL, or click: here.

To get a random quote you can either type "/api/random" at the end of this page's URL, or click: here.

To search a specific quote; type after the URL "/api/search/{_id}" and specifiy the id parameter. Ex: /api/search/KHA5folD-r.

Furthermore, add the /api/all suffix at the end of a search id param to return all quotes pertaining the queried param's character. Ex: /api/search/3zl9IjBc0/all.

Using the character {First_Name} will retrieve the first quote on their list, while using the {_id} will retrieve a specific quote of the character.

To single out a specific quote or retrieve a range of quotes you can use: /api/search/Kushina/quote/5 or: /api/search/B6GQsu6PYK6/quote/5-7.
